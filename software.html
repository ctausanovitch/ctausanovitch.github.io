gpuideal

gpuideal is an R package for fast Bayesian estimation of ideal point or IRT models. It was developed by myself and Jeff Lewis. gpuideal outperforms existing software by parallelizing the data augmentation algorithm and implementing it on NVIDIA GPUs. For more information about gpuideal, see our white paper. To use the package see the README on the GitHub page.
